# Mac-ubuntu
This will transform Linux mint 19.3 to Mac ubuntu

Install : 

sudo apt update 
sudo apt install xterm

Save MACOS to your desktop (Do not copy this to command line) 

cd Desktop 
chmod +x MACOS.sh 
./MACOS.sh
